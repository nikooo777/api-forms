# API-form

Basic API frontend in NodeJS

##Required dependencies
* NPM
* nodemon
* body-parser
* config
* cookie-parser
* debug
* express
* hbs
* morgan
* request
* server-favicon

## Installing dependencies


1. Install NodeMon globally ```sudo npm install -g nodemon```
2. Install the rest of the dependencies in node_modules ```npm install```
